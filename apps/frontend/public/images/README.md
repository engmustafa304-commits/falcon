# images

Public Falcon image assets belong here. Files in this folder are served from `/images/...`.

## Hero

Place homepage and public landing hero assets in `hero/`.

- `hero/hero-background.webp` - primary public homepage hero image.
- `hero/hero-placeholder.webp` - optional fallback hero image.

## Logo

Place raster showroom or Falcon logo assets in `logo/`.

- `logo/logo-falcon.webp` - Falcon showroom or hero logo asset.

## Cars

Place marketplace vehicle images in `cars/`.

- `cars/car-01.webp`
- `cars/car-02.webp`
- `cars/car-03.webp`
- `cars/car-04.webp`
- `cars/car-05.webp`
- `cars/car-06.webp`

## Dealers

Place dealer showroom images or logos in `dealers/`.

- `dealers/dealer-01.webp`
- `dealers/dealer-02.webp`
- `dealers/dealer-03.webp`
- `dealers/dealer-04.webp`

## Dashboard

Place dashboard preview screenshots in `dashboard/`.

- `dashboard/dashboard-owner.webp`

## Brands

Place vehicle brand marks in `brands/`.

## Categories

Place category imagery in `categories/`.

## Backgrounds

Place generic product or atmospheric background assets in `backgrounds/`.

If any image is missing, UI components must continue to render clean placeholder fallbacks.
